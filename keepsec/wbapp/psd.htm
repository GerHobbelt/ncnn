@webtoon/psd

A lightweight Adobe Photoshop .psd file parser in typescript with zero-dependency for browsers and NodeJS.

Installation

$ npm install @webtoon/psd --save

Browser compatibility

Features

  • Parse each layer information (offset, size, image data, etc)
  • Parse string value in text layer
  • Parse guides and slices
  • Supports .psb file
  • Supports unicode text (ex, layer title)

Benchmarks

against other PSD parsers on any PSD file.
